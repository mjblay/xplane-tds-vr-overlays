# Object Library

This folder will contain the TDS GTNXi screen-only overlay OBJ files.

## Required base variants

- TDS_GTN750_ScreenOnly_U1.obj
- TDS_GTN750_ScreenOnly_U2.obj
- TDS_GTN650_ScreenOnly_U1.obj
- TDS_GTN650_ScreenOnly_U2.obj

## Required TDS mappings

| File | TDS unit | Device |
| --- | --- | --- |
| TDS_GTN750_ScreenOnly_U1.obj | TDSGTN750XI_U1 | TDS_GTN750xi |
| TDS_GTN750_ScreenOnly_U2.obj | TDSGTN750XI_U2 | TDS_GTN750xi |
| TDS_GTN650_ScreenOnly_U1.obj | TDSGTN650XI_U1 | TDS_GTN650xi |
| TDS_GTN650_ScreenOnly_U2.obj | TDSGTN650XI_U2 | TDS_GTN650xi |

## Plane Maker object flags

When installed into an aircraft, these overlay objects should use:

- clickable: yes
- internal cockpit: no
- external cockpit: no

## Compatibility variants

Aircraft-specific compatibility variants may include hidden/dummy manipulator stubs to satisfy aircraft rconfig.txt validation.

The first known compatibility target is the Thranda F33A.
